개발 분야

MAMA는 아래와 같이 ~~~ 을 개발합니다. 추가로 ~~~ 을 포함해 ~~~도 개발해드립니다.

Web

MAMA는 기본적으로 Spring Boot, django, MySQL, Oracle, MongoDB 기반의 백엔드 시스템으로 개발하며, 프론트엔드는 고객님의 요구사항과 적합한 디자인에 알맞게 최적화된 개발을 해드립니다. RESTful API Server 도 개발합니다.

Android / IOS App

MAMA는 Android 와 IOS 의 네이티브, 하이브리드 및 웹 앱을 모두 개발합니다.

Arduino / Raspberry Pi

저희는 심플한 하드웨어 중심의 개발 부터, IOT 에 점목하는 복합적인 개발까지 모두 커버합니다.

서비스

저희는 기본적으로 아래와 같은 서비스를 제공하고, 추가 조건에 부합하도록 개발해드립니다.

호환성

어느 기기에서나 호환되는 것은 저희의 기본 옵션입니다. 예를 들어, 스마트폰, 태블릿, PC 등의 화면 차이를 고려한 반응형 디자인으로 화면에 따라 최적화된 디자인을 감상하실 수 있습니다. 또 다른 예로, 웹 서비스와 모바일 앱이 같은 서버를 쓰더라도 각자에 맞게 최적화 된 데이터 전송 및 관리가 기본입니다.

인프라

퍼블릭 클라우드에 런칭하는 것을 기본으로 합니다. 합리적인 인프라 설계로 높은 확장성과 가용성을 달성하고, 트래픽 증감에 따른 서버 용량의 증감이 실시간으로 이루어지므로, 높은 운영 가성비를 달성합니다.

보안

네트워크와 로컬 시스템에서의 기본적인 보안 구축은 기본 옵션입니다. 예를 들어, 웹 개발에서는 기본적으로 XSS, CSRF, Sql Injection 에 안전하게 개발됩니다. 또한, 추가적인 옵션에 따라 OTP 인증과 같은 Multi Factor Authentication 부터 LDAP, VPN 사용 등 원하시는 조건에 맞게 개발해드립니다.

유연함

저희는 유지 보수에 좋은 구조로 소프트웨어를 개발합니다. 덕분에 향후 기능의 확장과 같이 요구사항이 바뀌어도 신속히 대응할 수 있습니다. 업계 표준 소프트웨어 디자인 패턴과 '클린 코드'를 염두에 두고 개발해 만일 유지 보수를 다른 업체에 맡기시더라도 개발이 용이합니다.

표준

호환성이나 보안에서 문제가 있는 비표준 기술, deprecated 된 API 등은 사용하지 않습니다. MAMA는 반 완성형 3rd party 솔루션 등을 사용하기 보단 표준 기술을 사용해 직접 구현해 최적화합니다.

진보

MAMA는 가장 앞선 기술을 고집합니다. 그리고 그것은 트렌드, 유지보수성, 성능, 심지어 미적인 아름다움까지 특별한 소프트웨어를 구현하는 것을 가능케 합니다.
